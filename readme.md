# Panchatantra VR Storytelling System

An AI-powered interactive storytelling system that brings Panchatantra stories to life using:

- Scene-based story engine
- AI character dialogue (Groq API)
- Speech-to-text interaction
- Text-to-speech character voices

## Features

- Interactive story progression
- Character-based AI responses
- Voice interaction system
- Modular backend architecture

## Tech Stack

- Python
- Groq API (LLM)
- SpeechRecognition
- pyttsx3
- sounddevice
